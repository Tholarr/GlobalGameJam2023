# Global Game Jam 2023

48 hours to make a video game on the theme "root"

Result : a 2.5D first-person shooter where you are trapped in undergrounds tunnels and shoot roots monsters with a branch, before they can reach you in the mind of a child
Increasing difficulty throughout the levels

-- CSFML Library

## Compiling
You will need the standard libC and Make.
Just execute the following command :

    make

## Usage
Execute the program with the following command :

    ./underoots
